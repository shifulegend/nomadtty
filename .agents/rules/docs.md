# Antigravity — Documentation Rules
<!-- adapter; canonical source: docs/ai/engineering-rules.md -->
<!-- last updated: 2026-06-20 -->

- Update `docs/ai/**` FIRST, then synchronise adapter files.
- Mistake found → `docs/ai/mistakes.md` immediately (newest entry first).
- Decision made → `docs/ai/decision-log.md` immediately with timestamp.
- Notable change → `docs/ai/change-trace.md` before the commit.
- Code comments: only when WHY is non-obvious. Timestamp workarounds.
- Do not comment WHAT code does.
- README.md must stay in sync with current toolbar layout and architecture diagram.
