# Antigravity Workflow: start-session
<!-- last updated: 2026-06-20 -->

## When to use
At the start of every development session, before planning or writing code.

## Steps
1. Read `gemini/GEMINI.md` (this tool's entrypoint).
2. Read `AGENTS.md` (portability layer, confirms non-negotiable rules).
3. Read `docs/ai/session-start-checklist.md` — follow every step.
4. Read `docs/ai/mistakes.md` — note entries relevant to today's task.
5. Read `docs/ai/decision-log.md` — note decisions that apply.
6. State session summary:
   - Active rules relevant to today
   - Mistakes to avoid
   - Decisions that constrain today's work
   - Assumptions (state explicitly)
7. Propose a plan before any code.
8. Note which docs need updating at session end (see `docs/ai/tool-sync-policy.md`).
