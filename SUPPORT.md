# Support

## How to get help

Open a [GitHub Issue](https://github.com/shifulegend/nomadtty/issues) with:

- Your OS and how you installed NomadTTY (Docker / `install.sh` / manual).
- The device and browser where the problem occurs.
- What you expected vs what happened.
- Relevant nginx error logs (`sudo journalctl -u nginx`) or browser console output.

## Support model

This is a personal open-source project maintained on a best-effort basis. There is no SLA or guaranteed response time.

## Security issues

Do not open public issues for security vulnerabilities. See [SECURITY.md](SECURITY.md).
