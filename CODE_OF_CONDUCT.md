# Code of Conduct

## Our standard

Be respectful, constructive, and welcoming. Harassment, personal attacks, and discriminatory language have no place in this project.

## Scope

This applies in GitHub issues, pull requests, discussions, and any project-related communication.

## Enforcement

Violations may be reported to the project maintainer via GitHub. Reports will be reviewed promptly. The maintainer reserves the right to remove comments, close issues, or ban contributors who violate this standard.

## Attribution

This Code of Conduct is adapted from the [Contributor Covenant](https://www.contributor-covenant.org), version 2.1.
