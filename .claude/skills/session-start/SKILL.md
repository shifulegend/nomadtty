# Skill: session-start
<!-- Claude Code skill — invoke at the start of every session -->
<!-- last updated: 2026-06-20 -->

## Purpose
Ensures every session begins with full project context, avoiding repeated mistakes
and redundant decisions.

## Steps
1. Read `CLAUDE.md` (this tool's entrypoint).
2. Read `docs/ai/session-start-checklist.md` — follow every step.
3. Read `docs/ai/mistakes.md` — scan all entries; note which apply to today's task.
4. Read `docs/ai/decision-log.md` — note decisions that constrain today's work.
5. State session summary:
   - Active repo rules relevant to today
   - Mistakes to avoid
   - Decisions that apply
   - Assumptions being made
6. Propose a plan before writing any code.
7. Read `docs/ai/tool-sync-policy.md` to know what needs updating at session end.
